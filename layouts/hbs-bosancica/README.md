# Bosančica (Litogram)

Corpus-native bosančica: the BASE layer serves the medieval western-Cyrillic corpus (designed for typing Povaljska listina), not a modern typist's muscle memory. No dedicated Vukovian digraph keys; historic letters (djerv, jat, yers, uk, dzelo, omega) on base. No OS source exists.

Generated from the verified litogram layout definition
(`layouts/hbs-bosancica-design.json`) by keymapper's gen-os-artifacts —
the installable files teach exactly the arrangement
[litogram.org](https://litogram.org) does. Full design rationale:
the layout's page on litogram.org/keyboard-layouts/ and
CUSTOM_LG_LAYOUTS.md in the litogram repository.

## Sources

- Medieval western-Cyrillic corpus — Povaljska listina (1250), Charter of Ban Kulin (1189), Humac tablet
- The Unicode Standard — Cyrillic and Cyrillic Extended-B (U+0400–047F, U+A640–A66F), combining titlo U+0483
