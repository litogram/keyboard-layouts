# Gothic (Litogram)

Wulfila's Gothic alphabet (astral block U+10330–1034A); designed whole-cloth — no OS keyboard source exists.

Generated from the verified litogram layout definition
(`layouts/got-design.json`) by keymapper's gen-os-artifacts —
the installable files teach exactly the arrangement
[litogram.org](https://litogram.org) does. Full design rationale:
the layout's page on litogram.org/keyboard-layouts/ and
CUSTOM_LG_LAYOUTS.md in the litogram repository.

## Sources

- Wright, Grammar of the Gothic Language
- Braune–Ebbinghaus, Gotische Grammatik
- Streitberg, Die gotische Bibel (Wulfila edition)
- The Unicode Standard — Gothic block U+10330–1034A
